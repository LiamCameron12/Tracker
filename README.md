# DMW Tracker

Digimon Masters World utility — price tracker, dungeon stats, inventory manager and store helper.

## Download

👉 **[Click here to download the latest version](https://github.com/LiamCameron12/DMW-Tracker/releases/latest)**

1. Click the link above
2. Download `DMW Base.exe`
3. Run it — no install needed

> **Windows warning?** Click **More info** → **Run anyway** (normal for unsigned apps)

## Features

- Live item price tracker with Tera values
- Dungeon run stats (runs/hr, cores/hr, Tera/hr)
- Inventory value calculator with grand total
- Store item helper with click-to-copy commands
- GitHub Gist price sync — prices stay up to date automatically
